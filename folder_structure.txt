1) Backend
bash : pip install -r requirements.txt
bash : python -m venv venv
bash : venv/Scripts/activate
venv : python -m uvicorn app.main:app --reload ( make sure you are not in app, but in resource_management)

2) Frontend
bash : pip install streamlit pandas openpyxl requests
bash : venv/Scripts/activate
venv : streamlit_app run app.py (ensure you are in streamlit_app folder)
